# Session Manager Quick Reference

*Print this page for use during the conference.*

## Before Each Talk

- [ ] Check that there are enough speaker gifts for your slot
- [ ] Verify the correct speaker sheet is on top
- [ ] Confirm time cards are in order (15/10/5/STOP)
- [ ] Know who your session emcee is
- [ ] Confirm session emcee is aware of their duties and ready before the talk

## Session Emcee Backup

If there is no session emcee assigned for a slot, **you are the session emcee**.

Review the [Session Emcee Instructions](session_emcee.md) so you're prepared to:

- Introduce the speaker
- Manage time cards
- Facilitate Q&A
- Handle any issues
