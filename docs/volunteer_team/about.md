## Overview of Volunteer Team Responsibilities

The Volunteer Team coordinates all volunteer activities for DjangoCon US, ensuring the conference runs smoothly with adequate staffing at all key areas.

### Team Size

1-2 Volunteer Chairs

### Primary Responsibilities

- **Recruitment**: Create and manage volunteer sign-up spreadsheets, actively recruit volunteers to fill all slots
- **Training**: Provide written resources and onsite orientation to prepare volunteers for their roles
- **Coordination**: Ensure all volunteer positions are staffed throughout the conference
- **Communication**: Send pre-conference instructions and post-conference thank-you messages

### Volunteer Roles Managed

| Role | Description |
|------|-------------|
| Registration Desk | Check in attendees, distribute badges, answer questions |
| Session Emcees | Introduce speakers, manage Q&A, track time |
| Session Managers | Coordinate session logistics, backup for session emcees |
| Sprint Mentors | Help attendees during contribution sprints |
| Swag Bag Stuffers | Prepare swag bags before the conference |
| Online Moderators | Monitor chat, enforce CoC, support online attendees |
